## [0.1.0] - 2025-01-25

### Added
- Multi-selection append improvements, workspace-root defaulting, and refreshed project overview header.
- Configurable output document path (`llmFileCompanion.output.filePath`), overview template modes, and `.llmignore` support.
- Gitignore-aware tree summaries, per-extension statistics for summary headers, and the standalone “Refresh LLM Doc Overview” command.
- Progress notifications, skip summaries, and overview regeneration even when no files are appended.
